# Code Explorer Deep Course

该命令用于完整生成《项目分析与学习文档》库。

执行要求：

1. 必须完整执行五阶段
2. 每一阶段都要维护 XML 状态文件
3. 阶段 2 需要按波次启动多个 agents
4. 不允许跳过 `architect` 和 `writer`
5. 阶段 4 必须包含 `reviewer` 质量审查

在组织任务时，优先围绕这些教学单元：

- 项目入口与配置
- 核心概念与数据结构
- 主执行链路
- 调度与状态流转
- 文档装配与质量门禁
- 扩展点与辅助材料

最终必须生成完整导航型文档库，并汇报：

- 总览入口
- 初学者路径
- 进阶路径
- 两个最值得学习的设计
