# commands

该目录存放 `code-explorer` 的 Claude Code slash commands。

当前命令：

- `code-explorer.md`：完整五阶段工作流入口
- `code-explorer-quick-tour.md`：快速导览模式
- `code-explorer-deep-course.md`：完整深度课程模式
- `code-explorer-status.md`：检查当前工作区状态
- `code-explorer-verify.md`：校验生成文档

这些命令不依赖外部 CLI。它们直接在 Claude Code 中调度 agents、读取源码、写入 `.code-explorer/` 产物目录并维护状态文件。
